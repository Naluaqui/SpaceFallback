## Sobre o projeto

O **SpaceFallback Data Intelligence** é uma proposta tecnológica voltada à transparência, confiabilidade e rastreabilidade de dados espaciais que impactam diretamente serviços essenciais na Terra.

A solução parte de um problema atual: muitos sistemas utilizados no dia a dia dependem de dados gerados, enviados ou interpretados por satélites. Isso inclui GPS, comunicação via satélite, dados climáticos, telemetria, telecomando, payload, rastreio orbital e informações ambientais derivadas de infraestrutura espacial.

Essa dependência não é apenas teórica. Em maio de 2024, a Terra foi atingida por uma tempestade geomagnética extrema, classificada como G5, o nível mais alto da escala da NOAA. Eventos desse tipo podem afetar satélites, GPS, comunicações, redes elétricas e serviços que dependem de infraestrutura espacial. A notícia reforça que fenômenos espaciais podem gerar impactos reais em sistemas usados na Terra.  
Fonte: [The Guardian — Weather tracker: geomagnetic storms trigger northern lights](https://www.theguardian.com/environment/article/2024/may/13/weather-tracker-geomagnetic-storms-northern-lights)

Além disso, estudos sobre a tempestade de maio de 2024 analisaram impactos no ambiente orbital, incluindo aumento de arrasto atmosférico em satélites de órbita baixa. Esse tipo de análise mostra que eventos de clima espacial podem afetar não apenas a observação do céu, mas também a dinâmica de satélites e a continuidade operacional de serviços espaciais.  
Fonte: [Satellite Drag Analysis During the May 2024 Gannon Geomagnetic Storm](https://arxiv.org/abs/2406.08617)

Quando dados espaciais chegam atrasados, incompletos, imprecisos, divergentes ou sem rastreabilidade, a consequência deixa de ser apenas técnica e passa a afetar pessoas, empresas e operações reais. Uma falha de dado espacial pode impactar entregas, ambulâncias, agricultura, defesa civil, cidades inteligentes, conectividade, logística e operações críticas.

O objetivo do **SpaceFallback Data Intelligence** é criar uma camada de inteligência capaz de organizar missões e iniciativas ligadas à confiabilidade dos dados espaciais, permitindo que essas informações sejam monitoradas, compreendidas e utilizadas com mais segurança.

A proposta busca responder três perguntas principais:

1. O dado espacial está confiável?
2. Quem será impactado se esse dado falhar?
3. Qual ação deve ser tomada quando o dado não for seguro para decisão?

Dessa forma, a solução contribui para que dados espaciais não sejam tratados como verdades absolutas, mas como informações que precisam de contexto, origem, status, impacto e responsabilidade.

No contexto desta entrega, o sistema em **Java** simula a organização dessas iniciativas por meio de missões espaciais sustentáveis, permitindo cadastrar, listar, buscar, alterar status e resumir projetos relacionados ao SpaceFallback.

## Problema escolhido

O problema escolhido é a **baixa transparência e confiabilidade de dados espaciais utilizados em decisões que impactam a vida na Terra**.

Atualmente, diversos serviços dependem de informações espaciais, mas nem sempre o usuário final ou a equipe responsável consegue identificar:

- de onde o dado veio;
- se ele está atualizado;
- se possui precisão suficiente;
- se está completo;
- se pode sustentar uma decisão automática;
- qual impacto uma falha pode causar;
- qual ação deve ser tomada em caso de baixa confiabilidade.

No contexto da Global Solution, esse problema se conecta diretamente à nova corrida espacial, ao uso de satélites e à aplicação de tecnologias espaciais para resolver desafios reais da sociedade.

O projeto propõe organizar missões espaciais sustentáveis relacionadas ao **SpaceFallback Data Intelligence**, como:

- **Data Trust Score:** cálculo do nível de confiança dos dados espaciais;
- **Data Passport:** registro de origem, status, precisão e rastreabilidade do dado;
- **Impact Layer:** identificação de quem pode ser afetado por uma falha no dado;
- **Fallback Action:** recomendação de ações de contingência quando o dado não é seguro para decisão.

## Projeto Java

Este repositório contém uma aplicação desenvolvida em **Java puro**, executada pelo console, para simular o cadastro, organização e consulta de missões espaciais sustentáveis relacionadas ao **SpaceFallback Data Intelligence**.

A aplicação permite que o usuário interaja com um menu, cadastre missões, consulte informações, altere status e visualize um resumo final das missões cadastradas.

---

## Funcionalidades implementadas

A aplicação possui as seguintes funcionalidades:

1. Cadastrar missão espacial;
2. Listar missões cadastradas;
3. Buscar missão por nome;
4. Buscar missão por área de impacto;
5. Alterar status da missão;
6. Cadastrar exemplos oficiais do SpaceFallback;
7. Exibir resumo final.

O sistema também possui uma introdução interativa, apresentando o contexto do problema, a proposta do SpaceFallback e o desbloqueio dos menus especiais após algumas interações do usuário.

---

## Resumo final do sistema

O resumo final exibe:

- total de missões cadastradas;
- quantidade de missões por status;
- missão com maior grau de prioridade.

Esse recurso atende ao requisito de organização e análise das missões cadastradas no sistema.

## Descrição das classes

### `ProjetoEspacial.java`

Classe abstrata que representa a base de um projeto espacial sustentável.

Ela concentra atributos comuns, como nome, área de impacto, objetivo, ODS, tecnologia, prioridade e status. Também possui métodos auxiliares de busca por nome e área de impacto.

Essa classe foi criada para aplicar o conceito de **classe abstrata** e servir como base para a classe `MissaoEspacial`.

### `MissaoEspacial.java`

Classe que representa uma missão espacial sustentável ligada ao SpaceFallback.

Ela herda de `ProjetoEspacial`, utilizando o conceito de **herança**, e sobrescreve o método `exibirDados()`, aplicando **sobrescrita de métodos** com `@Override`.

### `Main.java`

Classe principal da aplicação.

É responsável por executar o sistema no console, exibir a introdução, controlar o menu, receber entradas do usuário e chamar as funcionalidades principais.

## Como executar o projeto

1. Abra o **Eclipse**.
2. Clique em **File > New > Java Project**.
3. Crie ou importe o projeto `SpaceFallbackJava`.
4. Verifique se os arquivos estão na estrutura correta de pacotes.
5. Abra a classe:

```text
src/br/com/fiap/spacefallback/app/Main.java
```

6. Clique com o botão direito em `Main.java`.
7. Selecione:

```text
Run As > Java Application
```

8. Interaja com o menu exibido no console.
